# Kurumi Discord Bot

<img src="assets/kurumichibi.png" width="200" />

Kurumi is a multi-purpose Discord bot with:
- AutoMod
- Information commands
- Manager tools
- Miscellaneous features
- Moderator tools

Currently has **33 commands**, aiming for **100+ commands**.
